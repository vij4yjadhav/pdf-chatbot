## 🚀 Features
- 📄 Upload any PDF
- 💬 Chat with your document
- 🧠 Uses RAG (Retrieval-Augmented Generation)
- ⚡ Fast responses using Groq API
- 🎨 Clean Streamlit UI

---

## 🛠️ Tech Stack
- Streamlit
- LangChain
- FAISS
- HuggingFace Embeddings
- Groq API

---

pip install -r requirements.txt
